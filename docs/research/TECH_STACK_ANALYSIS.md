# Technical Stack Check

Our clone will be built on the user's requested stack:
- **Framework:** Next.js (App Router, React)
- **Styling:** Tailwind CSS v4
- **UI Architecture:** shadcn/ui primitives.
- **Icons:** Lucide React
- **Animations:** Framer Motion (already available in package.json)

*Matching Target Patterns:*
- The dark mode and utility classes fit perfectly into Tailwind v4 context.
- We will use local components for the Marquee and Cover Flow (can utilize Framer Motion or pure CSS).
- Glassmorphism is easily replicable using `backdrop-blur-` classes from Tailwind.
